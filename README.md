# create-html-elements

Scaffold a Vite project using [vite-plugin-html-elements](https://codeberg.org/nexusocean/vite-plugin-html-elements).

## Usage
```bash
npx create-html-elements
```

Choose between:
- Bare HTML & CSS
- HTML with Tailwind (includes Prettier)

## Documentation

Visit [htmlelements.dev](https://htmlelements.dev)
